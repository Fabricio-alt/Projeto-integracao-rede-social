Sobre

Este trabalho proposto pela cocreare academy é um web site resposivo com integração com rede social. O projeto foi realizado em fases pelo grupo composto por Carlos Alberto, Fabrício, Valéria e Wesley. A primeira fase consistiu na construção da base do código em HTML, depois ele foi estilizado com CSS e por fim, código JavaScript foi utilizado para implementação de responsividade. Cada membro pode ao longo do processo contribuir com seu conhecimento, praticar e desenvolver mais habilidades.
